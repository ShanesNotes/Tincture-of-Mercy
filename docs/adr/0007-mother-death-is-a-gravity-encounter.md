# Mother death is a gravity encounter

Mother's death is fixed, but the encounter remains mechanically meaningful. The player performs real care actions, learns the shared substrate, witnesses the limit of cure, and carries consequences forward; the scene functions like an inverse first boss where the purpose is not victory but establishing gravity.

Considered Options:
- Make mother saveable through correct play: rejected because it would turn the opening into recipe mastery and weaken the care-is-not-cure premise.
- Make the death a non-interactive cutscene: rejected because it would waste the opening's chance to teach mechanics through embodied action.
- Use a fixed-outcome interactive encounter: chosen because it teaches play while establishing the moral and mechanical stakes.
