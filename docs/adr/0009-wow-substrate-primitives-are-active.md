# WoW substrate primitives are active for combat RPG mechanics

The active direction uses the WoW-shaped substrate reference for combat/RPG primitives: attack tables, threat, timers, resources, item quality, loot tables, progression hooks, and risk/reward. These mechanics should be grounded in Tincture's world and shared event substrate, but they should not be softened into purely symbolic combat or stripped of normal RPG economy.

Considered Options:
- Treat the WoW research as loose inspiration only: rejected because it is the reason the substrate document exists and would leave agents guessing about combat feel.
- Import all WoW genre furniture wholesale: rejected because Tincture still needs its own scale, tone, actors, and presentation.
- Use WoW-shaped primitives as active substrate constraints: chosen because it gives agents concrete mechanics while preserving the mercy RPG frame.
