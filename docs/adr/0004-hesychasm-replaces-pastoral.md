# Hesychasm replaces Pastoral in active direction

Active v0.9 docs use **Hesychasm** for the discipline/register of prayerful attention, interior watchfulness, and sustained presence. **Pastoral** remains available only as a historical/source-doc alias when discussing older materials.

Considered Options:
- Keep Pastoral as the active label: rejected because `latent_paths` gives the more precise theological and practice-shaped direction.
- Use both labels interchangeably: rejected because it would keep polluting future agent context.
- Rename active docs to Hesychasm and preserve Pastoral as an alias: chosen because it clarifies the current direction without hiding source history.
