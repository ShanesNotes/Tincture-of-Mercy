# Distinguish voice registers from latent paths

Active docs reserve **voice registers** for copy-language modes such as folk, sanctioned, and sacred. **Latent paths** names the player learning/attention paths: Apothecary, Hesychasm, and Iconographic. This keeps source material that says "three registers" usable as provenance without making future agents confuse language presentation with growth/practice structure.

Considered Options:
- Call both systems registers: rejected because it creates avoidable ambiguity.
- Rename latent paths to skills: rejected because it points toward a skill-tree frame too early.
- Use voice registers and latent paths: chosen because it preserves existing UI language while matching the preferred `latent_paths` direction.
