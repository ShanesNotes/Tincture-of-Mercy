# Kalev portrait / character-card source art

These are non-runtime portrait/dialogue/card candidates derived from the preferred version-1 Kalev generation. They are intentionally separate from `sheets/` runtime sprites.

Current artifacts:

- `kalev_v1_fullbody_portrait.png` — full-body portrait/card source from the large right-side figure.
- `kalev_v1_dialogue_bust_256.png` — dialogue bust candidate, transparent 256×256.
- `kalev_v1_dialogue_bust_512.png` — larger dialogue/card bust candidate, transparent 512×512.

Usage note: keep this lane for dialogue UI and character cards. Do not downscale these directly into runtime sprites; runtime 64×96 assets should use the mini-sprite / Aseprite pipeline instead.
