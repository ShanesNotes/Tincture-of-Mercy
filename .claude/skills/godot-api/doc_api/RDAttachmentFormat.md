## RDAttachmentFormat <- RefCounted

This object is used by RenderingDevice.

**Props:**
- Format: int (RenderingDevice.DataFormat) = 36
- Samples: int (RenderingDevice.TextureSamples) = 0
- UsageFlags: int = 0

- **format**: The attachment's data format.
- **samples**: The number of samples used when sampling the attachment.
- **usage_flags**: The attachment's usage flags, which determine what can be done with it.
