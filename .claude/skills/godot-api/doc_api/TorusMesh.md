## TorusMesh <- PrimitiveMesh

Class representing a torus PrimitiveMesh.

**Props:**
- InnerRadius: float = 0.5
- OuterRadius: float = 1.0
- RingSegments: int = 32
- Rings: int = 64

- **inner_radius**: The inner radius of the torus.
- **outer_radius**: The outer radius of the torus.
- **ring_segments**: The number of edges each ring of the torus is constructed of.
- **rings**: The number of slices the torus is constructed of.
