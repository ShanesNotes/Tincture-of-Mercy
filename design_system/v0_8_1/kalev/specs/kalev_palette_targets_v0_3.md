# Kalev Palette Targets v0.3

Status: guide palette only. Final palette should be harmonized with the root design tokens and current scene palette.

## Palette intent

Kalev should carry the Ironwood palette on his body:

- cedar brown;
- damp wool gray;
- blue-charcoal shadow;
- leather rot-brown;
- worn linen;
- bone/cedar charm;
- one restrained Ember red accent.

## Runtime palette rule

Do not add more saturation to solve readability. Use stronger silhouette, cleaner clusters, contrast at coat/pouch/notebook boundaries, and restricted red as attention punctuation.

Ember red should not become a constant glow in idle.
