What the regimes change · same DOM, different theme class

IronwoodFolk register everywhere. Patient name in IM Fell English; symptoms in italic body; actions short verbs. The notebook reads as graphite on lined vellum. Patterned water rims the bottom of the world.
WittehavenSame data, sanctioned register. The patient becomes CASE 0044-B; symptoms become RAD STAGE III; the "Write the name" verb becomes FILE OUTCOME. Radii drop to 0; the world grid replaces water. This is meant to feel relieving.
ParadiseThe same panel acquires .halo-active. The IC·XC plate appears top-right only in post-P0 Paradise review mode. Ember verbs are no longer offered; only "Sit" and "Write the name" remain. Sacred disclosure is a class, not a sprite.
Mid-craftThe patient panel slides aside; the Tincture Wheel appears with the player's crafted polygon overlaid. Quality reads in tactile words — sound — bitter but clear, never +34 healing.

Component contracts

No numbers leakThe review demo keeps sample values in JS, but production UI maps through RegisterLookup.tactile(field_id, value, regime_id). State meters carry a state word as their right-hand label; raw floats never reach the player.
Ember is earnedEmber is the only saturated red on the page. It glows in the pouch slot and as a single button tint on the patient panel. Use is a 220ms warm flash → 600ms desaturation; not modeled here, but the slot and button are wired for the trigger.
Hand tremorWhen Pressure is high, add the .tremor class to the cursor and to the Tincture Wheel pointer. 90ms ±0.5px translateX keyframe; never on the wheel itself.
Numbness blurWhen Numbness is high, add .numb-blur to the patient name (filter: blur(0.6px)). Case labels stay crisp. The perceptual debuff is expressed in CSS, not in copy.
Lucide CDN substitutes for production pixel-art glyphs. IM Fell English / EB Garamond / Cinzel / Caveat / IBM Plex Mono are licensed Google Fonts standing in for bespoke type. See colors_and_type.css for tokens; this kit is HTML for design review, not a runtime.
